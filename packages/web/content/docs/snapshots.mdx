Arctic uses Gitâ€‘based snapshots to track your working tree state. This enables safe undo/revert and benchmarking workflows.

## What snapshots are for

- Undo AI changes
- Compare changes in a session
- Apply and revert benchmark results

## Where snapshots live

Snapshots use a separate Git repository:

```
~/.local/share/arctic/snapshot/<projectID>/
```

This is separate from your project's `.git` directory.

## View diffs

```bash
arctic debug snapshot diff <hash>
```

## Apply a snapshot patch

```bash
arctic debug snapshot patch <hash>
```

## Track current changes

```bash
arctic debug snapshot track
```

This creates a new snapshot of your current working directory.

## Toggle snapshots

```json
{
  "snapshot": true
}
```
